This project is a simple library management system written in Python. It provides basic management of books and users, supports searching, borrowing and returning books, and includes unit tests.

## Objective
To build a small library system that supports:
- Managing books and users (add/remove/list)
- Searching for books (by title, author or keyword)
- Borrowing and returning workflow
- Basic statistics  on library usage and a plot of borrowed books per user
